# AM Trade Management - API Documentation

Welcome to the API Documentation package. This section serves as an onboarding guide to understand the lifecycle of our endpoints, including their internal service orchestration and external resource footprint.

## Core Modules

* [Trade API](api/TradeApi.md) - Endpoints for managing and creating trade executions.
* [Portfolio Summary API](api/PortfolioSummaryApi.md) - Endpoints for portfolio performance and asset allocation statistics.
* [Trade Journal API](api/TradeJournalApi.md) - Endpoints for the trader's psychological journal and reasoning notes.
* *More modules coming...*

Each endpoint document details the file name, method name, service connection, and data sink (MongoDB/Kafka) to help new developers trace execution from the web controller down to persistence.
